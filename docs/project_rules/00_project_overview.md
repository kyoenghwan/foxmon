## 프로젝트 전역 제약 사항

- **배너 UI/UX 규칙**: 모든 배너(본문 가로 배너 및 사이드 세로 배너)는 반응형 동기화 시스템을 통해서만 크기 조절이 가능합니다. 임의로 단일 배너의 크기를 변경하는 것은 레이아웃 붕괴를 초래하므로 금지됩니다.

## 범용 지침과의 관계

- **절대 불변**: 8-Atom 체계, 8대 절대 원칙
- **우선 적용**: 이 폴더(`docs/project_rules/`)에 존재하는 프로젝트 특수 지침이 범용 지침(`.rules/`)보다 우선 적용됩니다.
- **충돌 해결**: 원자 순수성을 훼손하지 않는 선에서 프로젝트 지침의 레이아웃/비즈니스 제약을 최우선으로 준수합니다.
